在役软件

主要是win端的各类软件

软件名称 简介 必备指数
Edge 浏览器三件套 ⭐⭐⭐⭐
Chrome 浏览器三件套 ⭐⭐⭐⭐⭐
Firefox 浏览器三件套 ⭐⭐⭐⭐
印象笔记 全平台通用的笔记软件，桌面版的标签系统非常喜欢；内地版越来越臃肿了，什么功能都想加进去但都没调试很好的感觉 ⭐⭐
Microsoft 365 微软全套 ⭐⭐⭐⭐⭐
adobe 一般按需取用 Ps Lr Pr Ai Id 等 ⭐⭐⭐⭐⭐
MagicEXIF 修复照片时间信息的软件 ⭐⭐⭐⭐⭐
傲软录屏 效果不错的录屏软件，已入终身会员 ⭐⭐⭐
Imagine 压缩图片大小的软件 ⭐⭐⭐⭐
Duplicate Cleaner Pro 快速查找删除重复文件的软件 ⭐⭐⭐⭐⭐
Typora 好用的Markdown写作文件，但只能在相应终端使用 ⭐⭐⭐⭐
7-zip 每次必用的解压缩软件 ⭐⭐⭐⭐
iTunes 为了听歌安装的 ⭐⭐
Xshell 终端模拟软件 ⭐⭐⭐
Git 免费开源的分布式版本控制系统 Git 的客户端 ⭐⭐⭐⭐
Visual Studio Code 不用对着文本编辑器瞎改了😥 ⭐⭐⭐⭐
PicGo 终于成功配置了，好用！ ⭐⭐⭐

web端

软件名称 简介 必备指数
notion 全平台通用的笔记软件，新宠，但是不能数据自有及本地部署，对数据的放心度不高 ⭐⭐⭐⭐
ProcessOn 流程图、脑图等等都可以画，目前比较喜欢 ⭐⭐⭐

手机端

软件名称 理由
nplayer 连接nas十分顺畅，原生软件不能解码的视频也可以轻松打开，界面友好，功能强大
Microsoft Lens 根本不需要全能扫描王！！！
AutoSleep 三件套 感觉已经成为自己的生命状态监测仪
Strong 想做健身规划及笔记的时候测试了很多软件，这款真的简洁到位
少数派 数码产品早知道

弃用软件

win端

软件名称 理由
迅雷 每次一下载必定开始疯狂弹广告
Internet Download Manager 不知道是不是我的打开方式有问题，日常报错
Windows Terminal 跟着sp教程操作的，但电脑差点卡废了
Xmind 同步不友好，找到了全平台替代方案

Linux端

软件名称 理由
Calibre-web 界面交互实在太痛苦了，而且不能在线打开书库，还不如远程连接文件夹然后用各终端自带软件进行阅读
bookstack 朋友推荐的自建笔记平台，也是交互过于痛苦，md编辑窗口显示非常不友好，最终也没找到一个符合自己的使用场景

其他专业软件

建模及模型分析软件

软件名称 简介
Rhino 曲面建模一绝
SketchUp 草图大师，青春的回忆
GoogleMap 很强大
ArcGIS 做景观分析还是很棒的，不过目前能力范围获取的数据精度太低，要么根据cad生成模型也可，但实际应用受限
envi 绿地植被分析时候用过的一个软件
Fragstats 景观格局指数分析软件
